JQuery

Javascript原生的方法就是可以把function当做参数来传递，jquery继承了这个方式

<pre><code>
$(document).ready(function(){
    $('div').click(function(){
        $(this).hover(
            function(){
                //do some thing
            };
            function(){
                //do some thing
            } ;
        );
    });
});
</code></pre>
